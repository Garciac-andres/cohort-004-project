# CLAUDE.md

## Coding standards

This project's coding standards live in the **`coding-standards`** skill, not here —
that keeps them out of the default context and loads only the relevant rule when code
is being written or reviewed.

- Skill: `.claude/skills/coding-standards/SKILL.md` (routes to topic files)
- TypeScript rules: `.claude/skills/coding-standards/standards/typescript.md`

When you add a new standard, add it to the skill (under the matching topic file), not
to this file.
